# 🌤️ Clima Online

Aplicação web que permite buscar a previsão do tempo de qualquer cidade em tempo real.

---

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- API OpenWeather

---

## 🔧 Funcionalidades
- Busca de cidade
- Exibição de temperatura
- Clima atual
- Umidade
- Ícone dinâmico
